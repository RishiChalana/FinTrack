# **App Name**: FinTrack AI

## Core Features:

- Account Management: Add, edit, and manage multiple wallets and accounts with starting balances and multi-currency support.
- Transaction Tracking: Record income, expenses, and transfers with categories, payment methods, notes, tags, and receipt attachments.
- Budgeting: Create budgets by category or total monthly expenses, track progress visually, and receive expense alerts.
- AI Financial Assistant: Utilize a pre-trained model to answer natural language queries, summarize spending, give financial advice, and generate quick reports. This feature will incorporate tool use to improve its recommendations and responses.
- Reports and Analytics: Generate daily, weekly, and monthly financial analysis with charts and graphs; export to PDF, CSV, or Excel.
- Goal Setting: Create savings goals, track progress visually, and get AI-driven suggestions to achieve goals faster.
- Admin Dashboard: Provide an admin interface to manage user accounts, view logs, and oversee system health and app-wide analytics.

## Style Guidelines:

- Primary color: Deep sky blue (#3091ff) evokes trust and clarity, key for financial apps.
- Background color: Very light blue (#E5F0FF), providing a clean and unobtrusive backdrop.
- Accent color: Purple (#9130ff), chosen to be analogous, is a great fit for highlighting key actions and insights.
- Font: 'Inter', a grotesque sans-serif, is suitable for both headlines and body text because of its modern and neutral look.
- Use clean, crisp, and minimalist icons to represent accounts, categories, and actions. Follow FastBudget's icon styles.
